# Gestione Presenze (CLI) — Progetto Python

Applicazione **da riga di comando (CLI)** in **Python** per la gestione delle presenze in una o più classi/corsi.

Il progetto permette di:
- creare e gestire più **corsi/classi**
- creare e gestire i **partecipanti**
- iscrivere i partecipanti ai corsi
- registrare le **presenze giornaliere**
- rivedere e modificare le presenze in modo semplice e chiaro

I dati vengono salvati su file `db.json`.

--- .

## ✅ Requisiti

- Python **3.10+** 
- Ambiente: **WSL Ubuntu** 
- Nessuna libreria esterna richiesta (solo librerie standard Python)
